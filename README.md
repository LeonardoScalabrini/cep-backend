STRING_CONNECTION_DB=mongodb://127.0.0.1:27017/correios

## Dev quick start ##

1. Install packages
````
npm install
````

2. Install MongoDB
````
docker-compose up
````

3. Create .env file

4. Start locally
````
npm start
````

## Run the tests ##

1. Install packages
````
npm install
````

2. Install MongoDB
````
docker-compose up
````

3. Run tests
````
npm test
````
